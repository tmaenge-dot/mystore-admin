MyStore — Lightweight wholesale storefront and admin

The problem

  • Small retailers need simple ways to order wholesale goods
  • Catalog updates and product image updates are slow and technical
  • Inventory and bulk pricing are hard to communicate

Our solution

  • Minimal admin + storefront
  • Secure image uploads and optional server-side processing
  • Bulk pricing, cart persistence, and order creation

Key features

  • Safe filename generation & MIME allow-list
  • Server-side resizing (sharp) and orphan cleanup
  • Audit logging and simple file-backed persistence

Demo flow

  1. Admin logs in and uploads a product image
  2. Storefront updates to show the new image
  3. Customer saves cart, checks out, and receives order confirmation

Pilot & Pricing

4-week pilot to integrate catalog and optional S3 presigned uploads. Custom pricing available.
